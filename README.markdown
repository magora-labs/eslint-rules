# eslint-rules

**_A rule set for eslint_**

Style rules closely follow google's javascript style guide.

### Installation

`npm i mgl-eslint-rules`

### Using as configuration file

Create a `.eslintrc` with

```json
{
  "extends": [
    "./node_modules/mgl-eslint-rules/.eslintrc"
  ],
  "rules": {
    // add your rules here
  }
}

```
